# To-Do List Aplikatsiyasi 📝

Local Storage bilan to-do list dasturi. Web va Mobile versiyalari mavjud.

## ✨ Funksiyalar

- ✅ To-do qo'shish/o'chirish
- ✅ Tugallashni belgilash
- ✅ Local Storage'da saqlash
- ✅ Kategoriyalar
- ✅ Search/Filter
- ✅ Edit qilish
- ✅ Dark mode
- ✅ Mobile responsive

## 📁 Tuzilishi

```
todo-list-app/
├── web/                 # React Web App
│   ├── src/
│   ├── public/
│   └── package.json
├── mobile/              # React Native
│   ├── App.js
│   ├── app.json
│   └── package.json
└── README.md
```

## 🚀 Ishga tushirish

### Web:
```bash
cd web
npm install
npm start
```

### Mobile:
```bash
cd mobile
npm install
expo start
```
